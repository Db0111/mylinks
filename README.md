# 📌 MyLinks

링크를 단순히 저장하는 것을 넘어, **왜 저장했는지까지 함께 기록할 수 있는 개인 아카이빙 서비스**

---

## 🧩 Problem

평소에 좋은 글이나 참고 자료를 발견하면  
나와의 채팅방이나 메모 앱에 링크를 저장하는 경우가 많습니다.

하지만 시간이 지나면:

- 링크가 계속 쌓여 어디에 무엇이 있는지 찾기 어려워지고
- 저장한 이유를 잊어버려 다시 활용하기 힘들며
- 주제별로 정리되지 않아 필요한 정보를 빠르게 찾기 어렵습니다

👉 단순 저장이 아니라 **“맥락 있는 저장”**이 필요했습니다.

---

## 💡 Solution

MyLinks는 링크와 함께 **메모, 목적, 태그**를 저장하여  
나중에 더 쉽게 찾고 활용할 수 있도록 돕습니다.

- 단순 URL 저장이 아닌, **저장 이유와 맥락을 함께 기록**
- 태그를 통해 **주제별로 정리**
- 검색과 필터를 통해 **빠르게 재사용**

---

## 🚀 Features (MVP)

- 🔗 링크 저장 (URL, 제목, 메모)
- 📄 링크 목록 조회
- ✏️ 링크 수정
- 🗑 링크 삭제
- 🏷 태그 기반 정리 (추가 예정)
- 🔍 검색 기능 (추가 예정)
